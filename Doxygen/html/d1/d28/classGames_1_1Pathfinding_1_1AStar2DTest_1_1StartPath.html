The main entry point for the pathfinding routines. AstarNode2D is derived from AStar then the StarPath class creates an instance of AStar and uses AstarNode2D to override the methods in AStar.cs. Using Path method as an entry and return point from/to BotMe. Also StartPath is used to make maps and check limits as well as print the map out in a console if we use console apps.
