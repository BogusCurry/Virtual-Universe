Detailed Description

Mapping between string name and Level object

Mapping between string name and Level object. This mapping is held separately for each SmartInspect.Repository.ILoggerRepository. The level name is case insensitive.

Level SmartInspect.Core.LevelMap.LookupWithDefault ( Level  defaultLevel)

Lookup a named level from the map

Parameters
defaultLevelthe name of the level to lookup is taken from this level. If the level is not set on the map then this level is added
Returns
the level in the map with the name specified

Lookup a named level from the map. The name of the level to lookup is taken from the Level.Name property of the defaultLevel argument.

If no level with the specified name is found then the defaultLevel argument is added to the level map and returned.
