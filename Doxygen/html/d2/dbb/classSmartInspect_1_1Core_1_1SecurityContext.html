Detailed Description

A SecurityContext used by SmartInspect when interacting with protected resources

A SecurityContext used by SmartInspect when interacting with protected resources for example with operating system services. This can be used to impersonate a principal that has been granted privileges on the system resources.

Parameters

An IDisposable instance that will revoke the impersonation of this SecurityContext, or null

Impersonate this security context. Further calls on the current thread should now be made in the security context provided by this object. When the IDisposable result IDisposable.Dispose method is called the security context of the thread should be reverted to the state it was in before Impersonate was called.

Implemented in SmartInspect.Util.WindowsSecurityContext, and SmartInspect.Util.NullSecurityContext.
