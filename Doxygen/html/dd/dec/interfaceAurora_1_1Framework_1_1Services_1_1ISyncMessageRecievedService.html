MessageReceived Aurora.Framework.Services.ISyncMessageRecievedService.OnMessageReceived

This is fired when a message from the ISyncMessagePosterService has been received either by the IAsyncMessagePosterService for Region or the ISyncMessagePosterService for Server Notes on this event: This is subscribed to by many events and many events will not be dealing with the request. If you do not wish to send a response back to the poster, return null, otherwise, return a valid OSDMap that will be added to the response.
