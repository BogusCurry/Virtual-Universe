class  ICCProfile
 This class models the ICCProfile file. This file is a binary file which is divided into two parts, an ICCProfileHeader followed by an ICCTagTable. The header is a straightforward list of descriptive parameters such as profile size, version, date and various more esoteric parameters. The tag table is a structured list of more complexly aggragated data describing things such as ICC curves, copyright information, descriptive text blocks, etc. More...
