Detailed Description

Interface implemented by logger repository plugins.

Plugins define additional behavior that can be associated with a SmartInspect.Repository.ILoggerRepository. The PluginMap held by the SmartInspect.Repository.ILoggerRepository.PluginMap property is used to store the plugins for a repository.

The SmartInspect.Config.PluginAttribute can be used to attach plugins to repositories created using configuration attributes.
