Detailed Description

Interface used by the LogManager to select the ILoggerRepository.

The LogManager uses a IRepositorySelector to specify the policy for selecting the correct ILoggerRepository to return to the caller.

Creates a new repository for the assembly specified.

Parameters
assemblyThe assembly to use to create the domain to associate with the ILoggerRepository.
repositoryTypeThe type of repository to create, must implement ILoggerRepository.
Returns
The repository created.

The ILoggerRepository created will be associated with the domain specified such that a call to M:GetRepository(Assembly) with the same assembly specified will return the same repository instance.

How the association between Assembly and ILoggerRepository is made is not defined. The implementation may choose any method for this association.

Implemented in SmartInspect.Core.DefaultRepositorySelector, and SmartInspect.Core.CompactRepositorySelector.

LoggerRepositoryCreationEventHandler SmartInspect.Core.IRepositorySelector.LoggerRepositoryCreatedEvent

Event to notify that a logger repository has been created.

Event raised when a new repository is created. The event source will be this selector. The event args will be a LoggerRepositoryCreationEventArgs which holds the newly created ILoggerRepository.
