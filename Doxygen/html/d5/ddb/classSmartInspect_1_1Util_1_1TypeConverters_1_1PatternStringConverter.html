Detailed Description

Convert between string and PatternString

Supports conversion from string to PatternString type, and from a PatternString type to a string.

The string is used as the PatternString.ConversionPattern of the PatternString.

See Also
ConverterRegistry, IConvertFrom, IConvertTo
